# Vue JS Enterprise Boilerplate Docs

> This project is made with vuepress, a static site generator by the vue core team for docs and other sites.

Core includes:

- Vuex
- Vue router
- Progressive Web App Support

---

System Requirements:

- node: `10.10.1`
- npm: `6.7.0`

# Up and Running

1. Npm i
2. Npm serve
